# dotfiles
dotfiles.

## Prerequisites
- Using chezmoi. Please install it in advance.
- If you're using macOS, hit the following command.

```
brew install chezmoi
```

## Spinning up my environment

First `chezmoi init`.

```
chezmoi init git@github.com:noir-h/dotfiles.git
```

Apply my settings.

```
chezmoi apply
```
